bits
====
A sample repo to illustrate how one can build a php application using packages* from various php frameworks.

Packages used:
 - Slim Php
 - Twig
 - Zend Authentication
 - Zend Cache
 - Zend Session
 - Doctrine ODM
 - Mandril
 - Mailchimp
 - Pimple
 - PhpResque/Fresque
 - Guzzle HTTP
 
To see all package versions checkout the composer.json
